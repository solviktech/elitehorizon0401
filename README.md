# Elite Horizon Catering

Multilingual Next.js App Router website for Elite Horizon Catering.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## GitHub Pages export

This project supports a static export build for GitHub Pages.

```bash
STATIC_EXPORT=true GITHUB_REPOSITORY=solviktech/elitehorizon0401 NEXT_PUBLIC_SITE_URL=https://solviktech.github.io/elitehorizon0401 npm run build
```

The exported site is generated in `out/`.
